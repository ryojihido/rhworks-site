# RHworks - Official Portfolio Site
氷堂リョージ（Ryoji Hido）の公式ポートフォリオサイトのソースコードです。

## URL
- https://edenthedoors.com

## Tech Stack
- **Framework**: [Astro](https://astro.build/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Interactions**: [Alpine.js](https://alpinejs.dev/)
- **Hosting**: GitHub Pages

## Deployment
GitHub Actionsを使用して、`main` ブランチへのプッシュ時に自動的にデプロイされます。
